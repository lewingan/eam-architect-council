# Golden Questions for EAM Architecture Council Evaluation

## Q1: Work Order Scheduling
How should we architect a work order scheduling module that optimizes preventive maintenance scheduling across 500+ assets, considering resource availability, asset criticality, and production windows?

## Q2: Predictive Maintenance Integration
What is the recommended architecture for integrating IoT sensor data with our SAP EAM system to enable predictive maintenance on critical rotating equipment?

## Q3: Mobile Maintenance Execution
How should we design a mobile-first maintenance execution platform that allows technicians to receive, execute, and close work orders in the field with offline capability?

## Q4: Asset Hierarchy & Functional Location Design
What are the best practices for designing a functional location hierarchy in a multi-plant manufacturing environment that supports both top-down planning and bottom-up failure analysis?

## Q5: Spare Parts & MRP Integration
How should we architect the integration between maintenance planning and spare parts inventory management to ensure parts availability for scheduled maintenance while minimizing carrying costs?
